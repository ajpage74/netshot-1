# Netshot
Docker version of Netshot (https://github.com/netfishers-onl/Netshot)

Includes java, mysql, and netshot

## How to start
docker run -ti  breaktheweb/netshot

default web path: https://serverip:8443

* Login: netshot
* Password: netshot
